## LoRaWAN 기반 배터리 관리 시스템


## 💻 프로젝트 소개
배터리 셀 모듈의 상태를 모니터링하고 관리하는 시스템

  
### 📌 주요 기능
- 실시간 배터리 상태 모니터링
- 이상 상황 감지
- 데이터 로깅
